# 🚀 Shivani Portfolio Website

## 📌 Project Overview

This is my personal portfolio website developed using React.js and Vite to showcase my education, technical skills, projects, certifications, internships, achievements, and contact information.

The portfolio acts as my professional online resume and helps recruiters quickly understand my technical background, project experience, and career goals.

Live Website:
https://shivani-portfolio-eta.vercel.app/

GitHub Repository:
https://github.com/Malipolishivani330/shivani-portfolio

---

# 🎯 Objective

The main objective of this project is to create a modern, responsive, and professional portfolio website that:

- Showcases my skills and projects
- Highlights certifications and internships
- Provides recruiters easy access to my resume
- Demonstrates my frontend development skills
- Serves as my digital professional profile
